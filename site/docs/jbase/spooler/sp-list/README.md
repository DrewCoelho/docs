## Description

Displays spooler configuration. It may be called as:

```
SP-LIST {options}
```

where:


| Option | Description |
| --- | --- |
| J | Job description list only. |
| P | sends output to the printer. Default action is to display to terminal. |
| Q | Form Queue configuration list only. |
