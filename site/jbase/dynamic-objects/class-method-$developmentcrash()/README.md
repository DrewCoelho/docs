# Class Method: $development::crash()

<PageHeader />

## Description

The **crash()** method will cause the system to crash with an access violation.

## Syntax

```
$development::crash()
$development->crash()
```

## Arguments

None

## Return value

Never returns

## Example

```
crash_and_burn = $development::crash() ;* caveat emptor
```
