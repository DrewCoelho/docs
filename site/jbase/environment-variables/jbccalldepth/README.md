# JBCCALLDEPTH

<PageHeader />

## Description

Set the stack size for a CALL

## Values

Integer

## Default

499

## Setting

As per normal environment variables, must be set prior to performing a subroutine call.

## UNIX

```
JBCCALLDEPTH=1024
export JBCCALLDEPTH
```

## Windows

```
set JBCCALLDEPTH=1024
```
