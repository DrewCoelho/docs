# CLOSESEQ

<PageHeader />

## Description

The **CLOSESEQ** statement closes the file previously opened for sequential access. It takes the general form:

```
CLOSESEQ FileVar,
```

Where:

**FileVar** contains the file descriptor of the previously opened sequential file.
