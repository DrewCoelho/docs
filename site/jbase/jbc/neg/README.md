## Description

The NEG function is used to invert the arithmetic value of an **expression**. It takes the general form:

```
NEG(expression)
```

Where **expression** may evaluate to any numeric value.

An example of use may be as:

```
varA = 10
varB = 12
CRT NEG(varA - varB)
```

to display the value 2.
