# TIME

<PageHeader />

## Description

The **TIME** function returns the current system time. It takes the general form:

```
TIME()
```

It returns the time as the number of seconds past midnight, as illustrated by:

```
CRT "Time is ":OCONV(TIME(), "MTS")
```
