# jCL-DE

<PageHeader />

Displays the current value of LastError.

## Syntax

```
DE
```

## Notes

The DE command is mainly used when debugging your jCL programs.

## Example

If LastError contains 404]61^QLNUMSEL, the DE command will display:

```
404]61^QLNUMSEL
```
