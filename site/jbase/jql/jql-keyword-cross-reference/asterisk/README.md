# Asterisk  

<PageHeader />

Used in a jQL query to indicate multiplication.

Can also be used to indicate a comment in a [PAragraph](./../../../jcl/paragraphs/README.md)

## Example  

```
LIST BOOK_SALES TOTAL EVAL "QTY * PRICE" AS Value FMT 'R2#10' CAPTION "Total Value"
```
