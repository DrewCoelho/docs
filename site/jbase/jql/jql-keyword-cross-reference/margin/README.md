# MARGIN

<PageHeader />

The **MARGIN** keyword is used to define the width of the left margin in a report.

## Syntax

```
MARGIN value
```

where:

**value** is the width of the margin to be used.

## Example

```
LIST BOOK_SALES HEADING "'C'Book Sales Report 'D'          Page 'PL'" MARGIN 5
```
