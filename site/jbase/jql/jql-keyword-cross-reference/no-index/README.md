# NO-INDEX, NO.INDEX

<PageHeader />

The **NO.INDEX** keyword indicates to the query processor that any index which exist should be ignored in the processing of this query.

## Syntax

```
NO-INDEX
```

## Example

```
LIST BOOK_SALES WITH TITLE_ID EQ "200" NO-INDEX
```
