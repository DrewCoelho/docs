# REQUIRE-INDEX, REQUIRE.INDEX

<PageHeader />

The **REQUIRE.INDEX** keyword indicates that an index must be used during the processing of this query.

## Syntax

```
REQUIRE-INDEX
```

## Example

```
LIST BOOK_SALES WITH TITLE_ID EQ "200" REQUIRE-INDEX
```
