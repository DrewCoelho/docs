# PN845

<PageHeader />

## Description

Conversion is processed in a named-attribute

## Previous Release Behavior

A dictionary with a named attribute, **N(A-correlative)**, processed both the correlative and the conversion of the named-attribute.

## Current Release Behavior

Only the correlative is processed.
