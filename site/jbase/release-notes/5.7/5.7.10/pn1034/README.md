# PN1034

<PageHeader />

## Description

 jQL: A-correlative sum function ignored in math operations

## Current Release Behavior

Prior to this patch, an A-correlative such as:

```
A;1-S(2)
```

would always return a value of `1`, ignoring the `S(2)`.
