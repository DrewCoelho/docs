# PN1076

<PageHeader />

## Description

LISTF (aka LISTFILES) does not list "f" or "q" pointers.

## Previous Release Behavior

"f" and "q" pointers are not listed along with "F" and "Q" pointers.

## Current Release Behavior

"f" and "q" pointers are listed.
