# DB-STATUS

<PageHeader />

## Syntax

```
DB-STATUS {-tvwV}
```

Where:

-**a**      All Databases
-**t**      Display users inside a transaction.
-**v**      Verbose mode.
-**w**      Display users currently waiting for DB-RESUME.
-**V**      Very verbose mode.

This command allows the system administrator to determine the state of the database.
