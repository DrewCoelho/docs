## Description

The jBASE **SUBD** command subtracts two decimal numbers. It's general form is as:

```
SUBD Num1 Num2
```

where **Num1** and **Num2** are any decimal numbers.

## Note

> Decimal number **Num2** is subtracted from decimal number **Num1**.
