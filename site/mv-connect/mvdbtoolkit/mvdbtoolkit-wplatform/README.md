# MVDBTOOLKIT.WPLATFORM

<PageHeader />

MVDBTOOLKIT.WPLATFORM will return a OBJ object of the PLATFORM.CONFIG record.  

## Usage

Command

CALL MVDBTOOLKIT.WPLATFORM(PLATFORM.OBJ)

### Example

```BASIC
CALL MVDBTOOLKIT.WPLATFORM(PLATFORM.OBJ)
CALL WOBJ(PLATFORM.OBJ,"GET","mvtype",MVTYPE,"",RERR)
PRINT MVTYPE
```
