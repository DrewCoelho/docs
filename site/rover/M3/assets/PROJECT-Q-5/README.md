## Cost/Revenue

![](./PROJECT-Q-5.jpg)

| **Amount Type**|  Select "Actual" to display the actual cost and revenue
values for each task or "Budget" to display the budget amounts in the charts.

-  
**3D**|  Check this box to display the chart in 3 dimensional format.

**Cost Chart**|  Displays a pie chart of the budget or actual costs for each
task.

**Revenue Chart**|  Displays a pie chart of the budget or actual revenue for
the project.
